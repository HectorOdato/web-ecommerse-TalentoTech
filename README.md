# Tienda Online - Proyecto Pre entrega TalentoTech
Esta tienda e-commerse esta desarrollada con HTML y CSS permite visualizar productos,encabezado,listas,formulario y informacion de contacto,apartado de reseñas

Instrucciones o capturas de pantalla mostrando cómo usar la app.
## Uso
1. Ingresá al sitio.
2. Navegá por los productos.
3. Completa formulario de contacto

## Tecnologías
- HTML
- CSS (Flexbox, Grid, Media Queries)

## Características
- Logo y titulo de la web
- Lista de productos
- Validación de formularios
- Seccion de reseñas
- Informacion de contacto
- Diseño responsive
